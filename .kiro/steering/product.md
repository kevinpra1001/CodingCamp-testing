# Product Overview

**Todo Life Dashboard** is a client-side, single-page personal productivity web app. It combines four widgets into one minimal interface:

- **Greeting Widget** — live clock (HH:MM:SS), full date, and time-of-day greeting
- **Focus Timer** — 25-minute Pomodoro-style countdown with Start / Stop / Reset controls
- **To-Do List** — persistent task management with add, edit, complete, and delete
- **Quick Links** — user-defined shortcut buttons (up to 20) that open URLs in new tabs

All data is stored in the browser's `localStorage`. No backend, no server, no account required. The app must work when opened via `file://` directly in a browser and as a browser extension page.

Target browsers: Chrome, Firefox, Edge, Safari (latest stable versions).
